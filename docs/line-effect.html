Line Effect

Apply a gradient or shadow effect within <hr /> horizontal rule lines.

evo-line-fade-[thickness]

evo-line-fade-thin
evo-line-fade-medium
evo-line-fade-thick

Create an <hr /> tag, and apply an evo-line-fade-[thickness] class. Default 'fade' lines are translucent. Do not apply the evo-line class.


   <hr class="evo-line-fade-thin" />
   <hr class="evo-line-fade-medium" />
   <hr class="evo-line-fade-thick" />

evo-line-fade-[color]

evo-line-fade-black
evo-line-fade-gray
evo-line-fade-white

Create an <hr /> tag, and apply an evo-line-fade-[thickness] class, followed by an evo-line-fade-[color] class. Fade 'color' lines are opaque. Do not apply the evo-line class.


   <hr class="evo-line-fade-thick evo-line-fade-black" />
   <hr class="evo-line-fade-thick evo-line-fade-gray" />
   <hr class="evo-line-fade-thick evo-line-fade-white" />

Experiment with other colors, as in this CSS example.


   hr.evo-line-fade-pink {  
     background-image: linear-gradient(to right, rgba(255, 0, 255, 0), rgba(255, 0, 255, 0.5), rgba(255, 0, 255, 0)); 
   }
