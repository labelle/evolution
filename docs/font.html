evo-font

This sentence displays the base font. Expect skilled signwriters to use many jazzy, quaint old alphabets effectively.

evo-font class will apply the default font styles directly to an HTML element. Default styles for font-family, font-size, and line-height are applied globaly to the <html> tag in _global.scss. Styles can be adjusted in _variables.scss.
