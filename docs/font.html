Font

Font-family base style for HTML tags such as <p> and <h1>.

Base Font

evo-font

experience design made simple
for living design systems

Create an HTML tag suitable for displaying text such as <p> or <h1>, and apply the evo-font class. The default font family can be adjusted in _variables.scss. 


   <p class="evo-font"></p>

evo-font-reset

This sentence demonstrates the reset of styles.

Use the evo-font-reset class to remove font styles.


   <p class="evo-font-reset"></p>
