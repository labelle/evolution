Base Text

evo-text

Experience Design Made Simple for living design systems

Create an <p /> tag, and apply the css class evo-text, followed by an evo-line-[style] class. Default line size is 'medium'. The default gray color can be adjusted in _variables.scss. 


   <p class="evo-text" /></p>
