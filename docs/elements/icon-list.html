Icon List

Icon lists are more dependentent upon HTML and CSS resets than other elements, particularly to control indents.

evo-icon-list

  • list item 1
  • list item 2
  • list item 3
<ul class="evo-icon-list">
<li>list item 1</li>
<li>list item 2</li>
<li>list item 3</li>
</ul>

This removes the bullets from an unordered list and controls text styles, plus imports custom SVG icon images. Create a <ul> tag with the evo-icon-list class. Place <li> tags nested underneath. Variables can be adjusted in _variables.scss
