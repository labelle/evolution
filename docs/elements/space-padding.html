Space Padding

Create space inside of an element with simple padding in a range of sizes. You can also use padding-right-10-before to set padding between a bullet and link, useful if you're just setting an icon such as bullet-check.

evo-space-padding-thick 
<div class="evo-space-padding-thick"></div>

Place a <div> tag on the element for which you want negative space at the bottom inside portion, and apply the evo-space-padding-thick class.
