Create an HTML tag suitable for displaying text such as <p> or <h1>, and apply the font-adobe-[family]-[style] class. The default Adobe Edge font family 'stack' can be adjusted in /variables/_font.scss. 


   <p class="font-adobe-source-sans"></p>
   <p class="font-adobe-source-serif"></p>
   <p class="font-adobe-source-code"></p>
