This is a <ul> list with customized bullets. Bullet colors can bullet symbol character can be customized. Create a <ul> tag that contains <li> tags, and apply the evo-list-bullet class. Add the list-snowflake class to the <ul> to change the symbol character. Use the list-bullet-gray class make the bullets look more neutral. List variables can be adjusted in _variables.scss
