Line Shadow

Apply a shadow or gradient effect within <hr /> horizontal rule lines.

evo-line-[shadow]

evo-line-gradient-1
evo-line-shadow-1
evo-line-shadow-2

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-...-[number] class.


   <hr class="evo-line evo-line-gradient-1" />
   <hr class="evo-line evo-line-shadow-1" />
   <hr class="evo-line evo-line-shadow-2" />
