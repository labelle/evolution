Line Size

Change the size of <hr /> horizontal rule lines.

evo-line-[thickness]

evo-line-thin
evo-line-medium
evo-line-thick

Create an <hr /> with an evo-line-[thickness] class. Uses the html <br-width> property values of thin, medium, and thick.

<hr class="evo-line-thin" />
<hr class="evo-line-medium" />
<hr class="evo-line-thick" />
