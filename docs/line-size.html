Line Size

Change the size of <hr /> horizontal rule lines.

evo-line-[thickness]

evo-line-thin
evo-line-medium
evo-line-thick

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-[thickness] class. Uses the html <br-width> property values of thin, medium, and thick.


   <hr class="evo-line evo-line-thin" />
   <hr class="evo-line evo-line-medium" />
   <hr class="evo-line evo-line-thick" />

evo-line-[size]

evo-line-smallest
evo-line-smaller
evo-line-small
evo-line
evo-line-big
evo-line-bigger
evo-line-biggest

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-[size] class. The evo-line class by itself serves as the medium value.


   <hr class="evo-line evo-line-smallest" />
   <hr class="evo-line evo-line-smaller" />
   <hr class="evo-line evo-line-small" />
   <hr class="evo-line" />
   <hr class="evo-line evo-line-big" />
   <hr class="evo-line evo-line-bigger" />
   <hr class="evo-line evo-line-biggest" />
