Highlight

Basic styles for text selection and highlight.

Base Selection State

<div class="selection">
Pack my box with five dozen liquor jugs.
<div class="selection">Pack my box with five dozen liquor jugs.</div>

Global ::selection styles are set in _global.scss. Selection and highlight styles can be adjusted from _variables.scss.

If needed, directly apply a .selection class to set the global selection style to an HTML element.

Base Highlight

<div class="selection highlight">
Clone Pack my box with five dozen liquor jugs. Several fabulous dixieland jazz groups played with quick tempo. Expect skilled signwriters to use many jazzy, quaint old alphabets effectively. 
<div class="selection highlight">Pack my box with five dozen liquor jugs.</div>

Create a span to contain your text, and apply thehighlight class, when applied after any other styling, will improve the visual appearence. Global ::selection styles are set in _global.scss. Selection and highlight styles can be adjusted from _variables.scss.

If needed, directly apply a .highlight after a .selection class to set the global selection style to an HTML element.
