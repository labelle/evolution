Line

Change the style of hr horizontal rule lines.

Base Line

<hr />

hr tag default styles can be adjusted in _variables.scss.

evo-line-[bw]

evo-line-black
evo-line-white 
<hr class="evo-line-black" />
<hr class="evo-line-white" />

Create an hr tag with an .evo-line-[bw] class. Black and white values can be adusted in the _variables.scss file.

evo-line-[thickness]

evo-line-thin
evo-line-medium
evo-line-thick 
<hr class="evo-line-thin" />
<hr class="evo-line-medium" />
<hr class="evo-line-thick" />

Create an hr tag with an .evo-line-[thickness] class. Uses the html <br-width> property values of thin, medium, and thick.

evo-line-[style]

evo-line-solid
evo-line-dotted
evo-line-dashed
evo-line-double 
<hr class="evo-line-solid" />
<hr class="evo-line-dotted" />
<hr class="evo-line-dashed" />
<hr class="evo-line-double" />

Create an hr tag with an .evo-line-[style] class.
