Line

Change the style of <hr /> horizontal rule lines.

Base Line

evo-line

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-[style] class. Default line size is 'medium'. The default gray color can be adjusted in _variables.scss. 


   <hr class="evo-line" />

evo-line-[style]

evo-line-solid
evo-line-dotted
evo-line-dashed
evo-line-double

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-[style] class.


   <hr class="evo-line evo-line-solid" />
   <hr class="evo-line evo-line-dotted" />
   <hr class="evo-line evo-line-dashed" />
   <hr class="evo-line evo-line-double" />

evo-line-reset

Use the evo-line-reset class to remove hr line styles.


   <hr class="evo-line-reset" />
