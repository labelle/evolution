Line

Apply .evo-line to hr tags. Some lines will default to a line-width of 'initial', which is equal to border-top-width: 'medium' (e.g. evo-line, evo-line-dashed). The .evo-line classes are optimized for hr tags. Hr tags can also accept .evo-space classes. To apply border lines to divs, use .evo-box elements.

Usage

Combine classes for desired effect.

evo-line evo-line-gray evo-line-double
