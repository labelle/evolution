Line Style

Change the style of <hr /> horizontal rule lines.

evo-line-[style]

evo-line-solid
evo-line-dotted
evo-line-dashed
evo-line-double

Create an <hr /> tag with an evo-line-[style] class.

<hr class="evo-line-solid" />
<hr class="evo-line-dotted" />
<hr class="evo-line-dashed" />
<hr class="evo-line-double" />
