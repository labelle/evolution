Usage
The nesting order of Bootstrap grid classes is important. To create a simple 1 column grid.
  1. Create an HTML tag using <div> or <section>, and apply a container class.
  2. Nest another HTML tag inside and and apply a row class.
  3. Repeat and apply the col-xs-12 class.
