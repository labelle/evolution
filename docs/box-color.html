Box Color

Color box borders and lines for <div> tags. Some boxes will default to a line-width equal to border-width: 'medium' (e.g. box-gray and evo-box-dashed).

Create an <div />, and apply the css class evo-box-[side], followed by an evo-box-[side]-[neutral] class. Colors can be adusted in the _variables.scss file.


   <div class="evo-box box-white">
   <div class="evo-box-top box-top-white">
   <div class="evo-box-bottom box-bottom-white">
   <div class="evo-box-left box-left-white">
   <div class="evo-box-right box-right-white">
