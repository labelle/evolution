Process

A new creative process is possible with a design system.

  1. Understand the problem by interviewing stakeholders
  2. Research the client organization and overall landscape
  3. Workshop and collaborate to identify pain points and key users
  4. Strategize about key insights and a path forwards
  5. Seek inspiration to inform a different approach
  6. Sketch some ideas and layouts by hand
  7. Debate and redo until a cohesive solution forms
  8. Mock-up UX designs and user flows
  9. Swag content to bring the ideas to life
  10. Groom user stories for agile-development requirements
  11. Rapidly build HTML layouts using the Evo framework
  12. Create guidelines using the overall evolution system
  13. Upload assets for use across the systems
  14. QA and user acceptance to validate and deliver the designs
  15. User validation from testing and metrics

Brand Theming

Basic outline for using evolution to apply branding to your designs.

  1. Decide on a brand color and fonts
  2. Change the variables in Evo according to the brand
  3. Swap out images so they match your look and feel
  4. Choose icons that feel closer to your branding
  5. Inverse colors need to be defined
  6. Jumbo sized elements need to be curated if desired
  7. Create examples of the brand in-use in UI
  8. Document what you have now via the Evolution System site
  9. Deploy the site and invite your team to use it
  10. Perform QA and look for glitches plus errors
