Create an html tag to hold the text, such as <p> or <h1>, and apply the css class evo-text, followed by an evo-text-[style] class. Styles will apply even without the evo-text class.


   <p class="evo-text evo-text-bold"></p>
   <p class="evo-text evo-text-italic"></p>
   <p class="evo-text evo-text-capitalize"></p>
