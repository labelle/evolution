Evo

Image

Image container for <img /> tags. Images will span 100% of their container.

img

<img src="../../image/brand/fpo.jpg" />

Use the <img /> tag with a source URL.

evo-image

<img class="evo-image image-size-28 image-fpo" />

Apply the evo-image class as in the above example. Background images will automatically size and crop to fit. Works with <div> tags too.

image-fill

<div class="position-relative image-ratio-16-9">
<a href="image-fill.html" target="_blank">
<img class="image-fill" src="../../image/brand/fpo.jpg" /></a>
</div>

Apply the image-fill class as in the above example. This will stretch the image to fill the container edge-to-edge. Requires position-relative as a parent class, and an image source <src="../../image/brand/fpo.jpg">.

<a href="javascript:voide(0);"><img class="evo-image image-random image-fill" /></a>

Use the image-fill class by itself, without position-absolute as a parent class. This will stretch image from edge-to-edge of your screen as in this example.

image-reset

Use image-border-none to suppress border attributes. <img /> tags with an empty source src="" may display border lines, a box, or bevel. This is particularly true if you apply a link to an <img /> tag. In such cases, try using a <div> tag instead.
