Evo

Space Margin

Create space outside of an element with simple margins in a range of sizes.

evo-space-margin-[width]

evo-space-margin-thin
evo-space-margin-medium
evo-space-margin-thick 
<div class="evo-space-margin-thick"></div>

Place a <div> tag on the element for which you want negative space at the bottom on the outside, and apply the evo-space-margin-thick class.

margin-[number]

<div class="margin-10"></div>

Place a <div> tag on the element for which you want negative space at the bottom on the outside, and apply the margin-10 class. You can also use just the space-10 class which will only apply a height value.
