Line Color

Change the color of <hr /> horizontal rule lines.

evo-line-[neutral]

evo-line-black
evo-line-gray
evo-line-white

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-line-[neutral] class. Colors can be adusted in the _variables.scss file.


   <hr class="evo-line evo-line-black" />
   <hr class="evo-line evo-line-gray" />
   <hr class="evo-line evo-line-white" />

evo-line-color-[darkness]

evo-line-color-lightest
evo-line-color-lighter
evo-line-color-light
evo-line-color
evo-line-color-dark
evo-line-color-darker
evo-line-color-darkest

Create an <hr /> tag, and apply the css class evo-line, followed by an evo-[color] class, and then an evo-line-color-[darkness] class. Colors can be adusted in the _variables.scss file.


   <hr class="evo-line evo-gray evo-line-color-lightest" />
   <hr class="evo-line evo-gray evo-line-color-lighter" />
   <hr class="evo-line evo-gray evo-line-color-light" />
   <hr class="evo-line evo-gray" />
   <hr class="evo-line evo-gray evo-line-color-dark" />
   <hr class="evo-line evo-gray evo-line-color-darker" />
   <hr class="evo-line evo-gray evo-line-color-darkest" />

By default, colors have a translucent opacity that reveals any background underneath. To remove the transparency, add the evo-line-color-solid class.


   <hr class="evo-line evo-line-gray evo-line-color-solid evo-line-color-lightest" />
