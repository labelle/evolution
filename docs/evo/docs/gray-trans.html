Create an HTML tag suitable for displaying text such as <p> or <h1>, and apply a trans-gray-[number] class.

<p class="trans-gray-25"></p>
 
 

For translucent gray backgrounds, create a <div> tag and add the -bg suffix.

<div class="trans-gray-15-bg"></div>

inverse-trans-gray-[number]

Aa

 
 

For inverse grays text or inverse gray backgrounds, follow the above instructions but add the inverse- prefix.

<p class="inverse-trans-gray-15"></p>
<div class="inverse-trans-gray-15-bg"></div>
