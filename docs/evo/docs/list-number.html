This is an <ol> list with customized numbers. Number background colors can be customized for up to three levels of nested list. Create an <ol> tag that contains <li> tags, and apply the evo-list-number class. Add the list-number-counter-zero class to the <ol> to change the number format. Use the list-number-down class to scootch the numbers down a bit. List variables can be adjusted in _variables.scss
