Form Type

Commonly used form input types and utility classes. Input types include text, search, URL, email, and password. <input type="text"> is the browser default.

text

Legend 
<label>Text Label</label>
<div class="evo-space-1"></div>
<input type="text" name="text" placeholder="Text Placeholder" />

Apply a "type" to your input classes, such as this <input type="text"> tag.

<input type="text" name="text" placeholder="Evo Text Placeholder" class="evo-form form-padding form-box" />

Apply a "type" to your input classes, such as this <input type="text"> tag, along with the evo-form, form-padding, and form-box classes.

search

Legend 
<label>Search Label</label>
<div class="evo-space-1"></div>
<input type="search" name="search" placeholder="Search Placeholder" />

Create an <input type="search"> tag.

<input type="search" name="search" placeholder="Evo Search Placeholder" class="evo-form form-padding form-box box-square" />

Create an <input type="search"> tag, then apply the evo-form, form-padding, and form-box classes. The box-square class is added to insure squared corners on IOs devices.

<input type="search" name="search" placeholder="Evo Search Placeholder" class="evo-form form-padding form-box form-icon-search box-square" />

Add the form-icon-search class to apply this icon.

<input type="search" name="search" placeholder="Evo Search Placeholder" class="evo-form form-padding-left form-box form-icon form-icon-left form-icon-search box-square" />

Apply the form-padding-left and form-icon-left classes to align the icon to the left side of the field.
