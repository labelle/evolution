Form Type

Form elements sync with overall styling of evo text and colors. Use <input type="text"> tags. Add attributes to the HTML tags as needed, such as autofocus, required, or multiple.

Apply a "type" to your input classes, such as this <input type="text"> tag, along with the evo-form, form-padding, and form-box classes.
