Evo

Image Size

Image sizes for <img /> or <div> tags. Images will span 100% of their container. Useful for creating a rapid prototype.

Example Image Size

<img class="evo-image image-size-23 image-fpo" />

Combine the evo-image class with any image-size-[number] class. image-size-23 is nearly a 16:9 aspect ratio. image-size-32 is a square. image-square is a square.
