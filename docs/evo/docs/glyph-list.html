Glyph List

Highly customized unordered lists of bullets. Text Lists are more dependent upon HTML and CSS resets than other elements, particularly to control indents. For additional elements that use a list-bullet, see the link-bullet page. Pseudo classes for space :before and :after are available in space-padding and space-margin.
