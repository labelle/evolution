Organization

Generally, the overall project and source code is divided between an "SCSS" folder which contains the SCSS source files, and the "Docs" folder which contains the documentation and multimedia assets. The website is organized into these major areas.

  • Evolution - landing page
  • Evo - The Evo framework and element style guide
  • Brand - Optional brand guide and asset library
  • Design - Pattern library
  • System - Knowledge base
