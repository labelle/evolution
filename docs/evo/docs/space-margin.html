Space Margin

Create space outside of an element with simple margins in a range of sizes. Negative margin sizes are also available.

margin-[number]

<div class="margin-10"></div>

Place a <div> tag on the element for which you want negative space at the bottom on the outside, and apply the margin-10 class.
