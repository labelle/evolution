Evo

Image

CSS classes and utilities, for images contained within <img /> or <div> tags. Images will span 100% of their container.

img

<img src="../image/fpo.jpg" />

Use the <img /> tag with a source URL. To add a border line around the image, try using the evo-box box-1 classes.

evo-image

<img class="evo-image image-size-28 image-fpo" />

Apply the evo-image class as in the above example. Background images will automatically size and crop to fit. Works with <div> tags too. Try the image-fluid class for 'max-width:100%' responsive sizing.

Image background utility classes help adjust the positioning and fit of background images.

image-border-none experimental

Use the image-border-none class to suppress border attributes. <img /> tags with an empty source src="" may display border lines, a box, or bevel. This is particularly true if you apply a link to an <img /> tag. In such cases, try using a <div> tag instead.

For hover effects, mix layout classes, image classes, opacity classes, and transition classes, as in the above example. Note the reverse stacking order of element layers.
