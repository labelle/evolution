Box

Border 'box' styles for <div> tags. The default line-width will equal 'border-top-width: medium'.

evo-box

evo-box 
<div class="evo-box"></div>

Establishes border box structure. Apply the evo-box class to a <div> tag. Works with <img /> tags too. Default style can be adusted in _evo_variables.scss

evo-box-right 
<div class="evo-box-right"></div>

Establishes border box structure limited to one side. Apply an evo-box-[side] class such as evo-box-right, as in the above example. Works with <img /> tags too.
