Brand Shadow

'Brand' font-family 'stacks' for HTML tags such as <p> and <h1>.

Base Brand Font Stack

brand-font

Native experience design made simple
for living design systems

Create an HTML tag suitable for displaying text such as <p> or <h1>, and apply the brand-font class. The default brand font family 'stack' can be adjusted in /brand/_font.scss. 


   <p class="brand-font"></p>
