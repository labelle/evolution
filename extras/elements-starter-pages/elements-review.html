June 15th, 2017

Project Brief

Evolution Elements Library

Mission

Define the look and feel for the new 'Evo' web app, so that the designers and developers have a clear visual reference and creative direction.

Approach

Because of the short timeline, begin by assembling inspiration in Pinterest boards. We'll discuss the boards and agree on aesthetic. Then we'll move to design the visual language boards directly in-code. We'll refactor the final board into the 'Evolution Toolkit' after final approval. Limit to 1 day of labor.

Tasks

  1. Create a visual language board for the 'Evo' website.
  2. Utilize the Evolution Elements Library for rapid UI assembly and review with the team.
  3. Construct the board in HTML and post the results to the evo-board repository.
  4. Refine as needed.
  5. Refactor into the 'Evolution Toolkit'.
